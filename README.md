# Étude de Faisabilité – Analyse de Données

Projet académique réalisé dans le cadre d’une étude de marché basée sur la collecte et l’analyse de données utilisateurs.

## Objectif
Analyser les besoins utilisateurs concernant une application de location de matériel entre particuliers.

## Outils utilisés

- Google Forms
- Excel
- Analyse quantitative

## Compétences développées

- Analyse de données
- Visualisation de données
- Étude de marché
- Interprétation de résultats
- Dashboard Excel
